# Security Policy

## Supported versions

This project is currently in the **0.x** stage and moves quickly.
Security fixes are applied to the latest minor release.

## Reporting a vulnerability

If you believe you have found a security vulnerability in **torch-audit**, please report it privately.

- Email: **malkiv.roman@gmail.com**
- Subject: `torch-audit security report`

Please include:
- A clear description of the issue
- A minimal reproduction (if possible)
- The versions of torch-audit / torch / python

You can expect an initial response within a few days.

## Public disclosure

Please avoid filing public GitHub issues for security vulnerabilities.
